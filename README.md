# Портфолио с фильтрами

React приложение для отображения портфолио проектов с фильтрацией по категориям.

## Компоненты

- **Portfolio** (классовый) — главный компонент с state
- **Toolbar** (функциональный) — панель фильтров
- **ProjectList** (функциональный) — список карточек

## Запуск

```bash
npm install
npm run dev
```

## Фильтры

- All
- Websites
- Flayers
- Business Cards
